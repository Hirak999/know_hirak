Projects

Multilingual NMT featuring Llama

Finetuned Llama-2-7b-hf for multilingual neural machine translations, for conversion of Hindi and Bengali to English. This project is currently ongoing and support for more languages are on the way. Future plan is to support conversion from Language X to Language Y, instead of Language X to only English.
Check out this cool work at HuggingFace

Named Entity Correction using G2P Techniques

Named Entity Recognition is one of the most challenging task for an ASR system. This research focuses on finding effecient solution for Named Entity correction using Grapheme to Phoneme (G2P) techniques. This new method achieved accuracy at per with existing SOTA techniques.

Speech-to-Speech Translation using Fairseq

Instead of taking an cascaded approach where first Language X is converted to text in source language, then that text being translated to target language using NMT, and from there to speech in the target language, in this project we explore how to directly convert source speech to target speech. The project is based on Fairseq module devoloped by META. We have explored this project on French to English data having an BLEU score of 27 and Hindi to English data having an BLEU score of 30. Future plan is to improve the BLEU score and to support other languages.
